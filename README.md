# Prefactor Google Cloud Integration Website

This is a futuristic, tech-themed website showcasing Prefactor's AI Agent Identity Platform integration with Google Cloud. Built for GitHub Pages deployment.

## Features

- **Futuristic Tech Design**: Dark theme with neon blue accents, animated backgrounds, and dynamic visuals
- **Responsive Layout**: Fully responsive design for all device sizes
- **Interactive Elements**: Animated particles, rotating 3D cube, interactive integration diagram
- **Google Cloud Integration Showcase**: Detailed visualization of how Prefactor integrates with Google Cloud services
- **Contact Form**: Functional contact form with validation

## Setup for GitHub Pages

1. **Create a new GitHub repository** named `prefactor-google-cloud-site`

2. **Upload all files** to the repository:
   - `index.html`
   - `style.css`
   - `script.js`
   - Create `assets/` folder with your logo and images

3. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to "GitHub Pages" section
   - Select "main" branch as source
   - Click "Save"

4. **Your site will be live at**: `https://[your-username].github.io/prefactor-google-cloud-site/`

## Customization

### Colors
Modify CSS variables in `:root` section of `style.css`:
```css
:root {
    --primary: #00f3ff;          /* Neon blue */
    --primary-dark: #0080ff;     /* Darker blue */
    --secondary: #8a2be2;        /* Purple accent */
    --dark-bg: #0a0a1a;          /* Dark background */
    --darker-bg: #050510;        /* Even darker background */
}
